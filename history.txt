[UI 개선 - Defect 선택 하이라이트]

===========================================
주요 기능 요약
===========================================

[이미지 입력]
- Basler 카메라 연결
- 파일 로드 (다이얼로그)
- Drag & Drop 이미지 로드

[검사 기능]
- Start/Stop Inspection
- Threshold 조정 (0~255)
- Min Area 조정 (0~500)
- Adaptive Threshold 옵션

[결과 표시]
- Result 리스트: 검출된 Defect 목록 (분류, 면적)
- ImageView: 전체 이미지 + 빨간색 contour + 노란색 선택 하이라이트
- DefectView: 선택한 Defect의 확대 이미지 (원본 해상도, 최소 150x150)

[뷰어 조작]
- Ctrl+마우스휠: 확대/축소 (0.2x ~ 6.0x)
- 좌클릭+드래그: 패닝 (이미지 이동)
- Defect 클릭: 해당 Defect 선택 및 상세 보기

[성능 최적화]
- 원본 해상도 유지로 선명도 보장
- ROI 크롭 방식으로 빠른 렌더링
- 비연속 배열 처리로 안정성 확보
